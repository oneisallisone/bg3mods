## Getting Started

### Prerequisites

```bash
node.js >= 18.0.0
npm >= 9.0.0
```

### Installation

1. Clone the repository:
```bash
git clone https://github.com/oneisallisone/bg3mods-git.git
cd bg3mods
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
